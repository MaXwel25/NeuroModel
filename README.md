***NeuroModel***

NeuroModel - это учебный проект по созданию искусственной нейронной сети на языке C#. Основная задача проекта - распознавание рукописных цифр (0–9) на основе входных данных, представленных в виде матрицы пикселей (3 на 5).

Проект разработан в рамках учебного курса и демонстрирует понимание основных принципов работы нейронных сетей: прямого распространения сигнала, обратного распространения ошибки (Backword pass) и процесса обучения с использованием градиентного спуска.

В качесте функции активации выступает **Гиперболический тангенс**.

Датасет находися в файле test и представляет собой массив чисел, которые поступают на анализ сети.
